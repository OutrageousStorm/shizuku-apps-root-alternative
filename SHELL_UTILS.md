# Shizuku Shell Utilities

Apps that use Shizuku to provide root-like shell access for scripting and automation.

## System shell access

### App Manager
- Full system command execution
- APK extraction
- Permission management
- Uninstall system apps (without root)
- [GitHub](https://github.com/MuntashirAkon/AppManager)

### Termux (with Shizuku plugin)
- Full Bash shell via Shizuku integration
- Run scripts with elevated ADB permissions
- Package manager for Linux utils
- F-Droid: Termux + Termux:Styling

### Adb-Idea (Android Studio plugin)
- ADB IDE integration for developers
- Execute commands from IDE
- Deploy APKs directly

## Task automation

### Tasker (with Shizuku Plugin)
- Automate system tasks based on triggers
- Execute shell commands on events
- Create complex workflows
- Play Store

### MacroDroid
- Visual automation without Tasker's learning curve
- Shizuku support for system-level control
- F-Droid available

## File management + system access

### MiXplorer + Shizuku Module
- System file access
- Backup system partitions
- Browse /system and /vendor directly
- F-Droid: MiXplorer

## Performance tuning via Shizuku

- **Turbo Boost** — CPU freq scaling
- **Thermal Guardian** — temperature monitoring
- **Performance Control** — governer switching

Most require Shizuku permission grant in the Shizuku Manager app after install.
