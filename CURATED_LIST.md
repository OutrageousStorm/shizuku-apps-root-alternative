# Shizuku Apps — Curated List (Updated May 2026)

The best no-root tools powered by Shizuku's ADB-level access.

## System & Settings

| App | Link | What it does |
|-----|------|-------------|
| **Shizuku** | [GitHub](https://github.com/RikkaApps/Shizuku) | The bridge itself — required |
| **App Manager** | [GitHub](https://github.com/MuntashirAkon/AppManager) | Full app control — disable, backup, perms |
| **Shelter** | [F-Droid](https://f-droid.org/app/net.typeblog.shelter) | Isolate apps in a work profile |
| **NetGuard** | [F-Droid](https://f-droid.org/app/eu.faircode.netguard) | Per-app firewall (no root) |
| **Island** | [Play Store](https://play.google.com/store/apps/details?id=com.oasisfeng.island) | App isolation + cloning |
