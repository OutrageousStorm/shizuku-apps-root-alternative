| App | Function | Source |
|-----|----------|--------|

| Blocker | Block components, apps, intents | [F-Droid](https://f-droid.org/packages/com.samrudhz.blocker) |

## Automation

- **Tasker** — Automation scripts with Shizuku integration
- **MacroDroid** — Task automation without root
- **Automate** — Visual automation builder

## Privacy

- **Watomatic** — Auto-reply to messages
- **TrackerControl** — Block trackers without root

## Best Combo

Install Shizuku + App Manager + Shelter for:
- Full app control (disable, uninstall, backup)
- Work profile isolation
- No root needed
